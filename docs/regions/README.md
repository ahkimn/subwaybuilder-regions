# Regions Mod

[Back to root README](../../README.md)

**SubwayBuilder Regions** allows users to import real-world geographic regions (e.g. ZIP codes, counties, wards) into the game SubwayBuilder, adding a visualization layer on top of the in-game map as well as additional panels for region-based statistics such as population, commuter flows, and infrastructure.

## Build

```bash
npm run build:regions
```

## Development

```bash
npm run build:dev
```

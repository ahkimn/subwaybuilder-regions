# Enhanced Demand View Mod

[Back to root README](../../README.md)

**Enhanced Demand View** enhances the in-game demand visualization in SubwayBuilder.

Releases are published to the sister repository [`subwaybuilder-enhanced-demand-view`](https://github.com/ahkimn/subwaybuilder-enhanced-demand-view).

## Build

```bash
npm run build:enhanced-demand-view
```

## Development

```bash
npm run build:dev:enhanced-demand-view
```
